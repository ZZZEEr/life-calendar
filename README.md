# Life Calendar 生命日历壁纸

飞牛NAS / Docker 一键部署的手机壁纸日历，支持自定义颜色、纪念日、背景图。

## 部署方式（飞牛NAS推荐）

1. 在飞牛文件管理器新建文件夹 `/docker/life-calendar`
2. 把本仓库文件上传/ git clone 到该文件夹
3. 把你的字体文件（AppleColorEmoji.ttf + 所有 ARIAL*.TTF）复制到文件夹里
4. 在 Docker → Compose → 新增项目 → 选择该文件夹 → 立即启动

访问：http://你的NAS-IP:811

字体版权：Arial 和 AppleColorEmoji 为商业字体，请自行从电脑复制，不要上传 GitHub。
